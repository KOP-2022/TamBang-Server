# TamBang-Server
TamBang 애플리케이션의 WAS를 구현한 프로젝트 repository 입니다.

Java11, Spring boot framework를 이용해 개발 중입니다.  
MySQL RDBMS를 database로 활용하고, JPA ORM을 이용해 DB와 데이터를 교환합니다.
